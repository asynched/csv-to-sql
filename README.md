# CSV-TO-SQL

This is a CLI application that converts a CSV file to an SQL query.

## About

### Building

#### Requirements

1. `rustc`
2. `cargo`

#### Running the app (development)

```sh
$ cargo run
```

#### Building the app (production)

```sh
$ cargo build --release
```
